# Pasos


**1. git commit -m**

Primero se realizan los dos primeros commits A y B en la rama main

**2. git branch exp**

Se crea una nueva rama llamada exp

**3. git checkout exp**

Me muevo a la rama exp y creo el commit C

**4. git checkout main**

Me muevo a la main y creo el commit E

**5. git checkout exp**

Me muevo a la rama exp y creo por último el commit D

**6. Por último se realiza un merge de la rama exp a la main desde el IDE**

Se hace el último commit con el Readme
